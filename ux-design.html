01 ux design

UX设计 / 动态设计 / 交互体验 / 实验研究

从版式中的信息层级、色彩之间的呼应，到界面中的反馈变化与交互体验，我开始关注设计与人之间的联系。

于是，设计的重心也逐渐从视觉表现延伸到用户体验研究。我开始重新观察那些容易被忽略的细节：一次短暂的停留、一个细微的反馈、一段等待中的变化。那些看似微不足道的设计元素，往往正在悄悄影响人们的情绪、注意与行为。于是设计实验构建网站并通过募集用户测试，观察动态设计如何影响人们的使用体验，以及人与界面之间的互动关系。

通过不断地尝试，观察与验证，我开始从用户的视角重新理解设计，并持续探索如何通过交互设计为用户带来更自然、更细腻且富有温度的体验。

02 web design

Web设计 / 动态设计 / 图标设计 / 毕业设计

「日日是好日」 源于一句禅语。意为生活的好坏并非绝对，而取决于人的心境与对事物的理解。

本网站以“时间与自然”为主题，由五个部分构成。深蓝象征静谧的夜晚，暖黄象征光亮与温度。即使身处漫长黑夜，也有月光相伴，而太阳终将在新的一天升起。页面中的箭头符号象征时间不断向前流动，并充满未知与可能。十二个月份则分别选用对应时节的植物作为图标，以呼应四季更迭与自然规律，表达顺应自然、感受时间流转的设计理念。

一月一物，一周一色，一日一签。

希望使用者带着一点期待打开它，在浏览的过程中，重新发现日常生活中那些容易被忽略的美好，日常生活也会变得多样多彩吧。
